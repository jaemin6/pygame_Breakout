# 🧱 Pygame Breakout (벽돌깨기 게임)

## 📌 개요 (Overview)
이 프로젝트는 **Python의 Pygame 라이브러리**를 사용하여 구현된 고전 아케이드 게임 **벽돌깨기(Breakout)**입니다.  
패들(Paddle)로 공을 튕겨 벽돌을 파괴하고, 다양한 **파워업 아이템**으로 더욱 재미있게 플레이할 수 있습니다.

---

## 🕹️ 주요 특징 (Key Features)

### ✔ 다양한 스테이지
- 서로 다른 벽돌 배치 패턴을 가진 **여러 스테이지** 제공  
- `STAGES` 배열 구조로 쉽게 스테이지 추가 가능

### ✔ 파워업 아이템
벽돌을 깨면 일정 확률로 아이템이 떨어지며, 다양한 효과를 제공합니다.

| 아이템 색 | 이름 (Item) | 효과 (Effect) |
|----------|-------------|-------------------------------|
| 🟡 노란색 | **Extend** | 패들의 길이를 길게 확장 |
| 🟢 초록색 | **Life** | 생명 +1 증가 |
| 🔵 파란색 | **Multi** | 공 추가 → 멀티볼 생성 |
| 🔴 빨간색 | **Speedup** | 공의 이동 속도 증가 |
| ⚪ 회색 | **Missile** | 미사일 모드 활성화 (Space 바로 발사 가능) |

### ✔ 미사일 모드 (Missile Mode)
- **Missile 아이템 획득 시 활성화**
- 스페이스바(Space)를 눌러 미사일 발사  
- 미사일로 벽돌을 직접 파괴 가능
